git rash 桌面版本地提交代码主要命令

$ git --help 帮助命令

$ git pull origin master 将远程仓库里面的项目拉下来

$ dir  查看有哪些文件夹

$ git rm -r --cached target  删除target文件夹
$ git commit -m '删除了target'  提交,添加操作说明

cd /D   //进入指定盘

git clone (创建的github的库的网址)    //将github的库创建到本地

git add （文件名.格式名）  //提交

git commit -m "*"          //给上传的文件的做备注，方便查找记忆

git push origin master     //登陆
